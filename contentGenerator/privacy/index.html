Privacy Policy

Last Updated: 04.03.2025

1. What Data Do We Collect?

Our extension only processes user prompts, model responses, and optionally the page content. These data are temporary and not stored.

2. Purpose of Data Usage

The collected data is used to generate text based on user requests and to help the model understand the context. It is not shared with third parties.

3. Data Storage and Security

Data is processed only during the session and not stored. All data transfers are encrypted via HTTPS.

4. User Rights and Choices

Users who do not want their prompts to be processed by the model may refrain from using the extension.

5. Third-Party Services

The extension only uses GroqCloud API and does not share data with any other services.

6. Changes and Updates

This privacy policy may be updated periodically. Updates will be notified within the extension.
